Assistant d'import Codex

Ouvre l'assistant depuis ce bouton. Une fois dedans, regarde le panneau Import initial guide en haut a gauche.

Ouvrir l'assistant d'import
  1. Clique sur Demo import en haut de la page, ou sur Exemple dans le panneau d'import.
  2. Lis la phrase verte : elle te dit exactement quoi faire ensuite.
  3. Clique sur Analyser le fichier ou le collage.
  4. Clique sur Utiliser ces donnees pour optimiser.
